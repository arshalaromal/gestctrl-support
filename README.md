# GestCtrl Support

Official support and issue tracker for GestCtrl.

Use this repository to:

- Report bugs
- Request features
- Ask questions
- Share feedback

---

## Before Creating an Issue

Please:

- Search existing issues first
- Make sure you are using the latest version of GestCtrl
- Include clear reproduction steps
- Attach screenshots or recordings if possible

---

## Bug Reports

When reporting a bug, include:

- App version
- Operating system
- Device model
- What you expected to happen
- What actually happened
- Steps to reproduce the issue

---

## Feature Requests

Feature suggestions are welcome.

Please explain:

- What you want added
- Why it would be useful
- Any similar apps/features for reference

---

## Important

This repository is only for public support and issue tracking.

GestCtrl itself is closed-source proprietary software.
